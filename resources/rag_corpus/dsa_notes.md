# DSA Quick Notes (Seed Corpus)

## Binary Search Tree

A Binary Search Tree stores keys so that left subtree keys are smaller and right subtree keys are larger than the root key.
Average search, insert, and delete are O(log n) for balanced trees. In skewed trees, operations can degrade to O(n).

## Dynamic Programming

Dynamic Programming solves problems with overlapping subproblems and optimal substructure.
Two common approaches are memoization (top-down) and tabulation (bottom-up).

## Graph Traversal

Breadth-First Search explores level by level and gives shortest path in unweighted graphs.
Depth-First Search explores depth-first, often implemented with recursion or an explicit stack.
