# Resources

This folder contains reusable learning assets for the 5-day Agentic AI course.

## Structure

- `references/`: day-wise further-study links (trusted channels, <= 2h each topic)
- `prompt_templates/`: copy-ready prompts used in examples
- `qa_lane/`: evaluation resources for QA engineers transitioning to AI
- `rag_corpus/`: small source documents for local Chroma RAG demos
- `study_notes/`: generated notes and student reflections
- `study_reports/`: reports generated by autonomous agents
- `vector_store/`: local Chroma persistence directories

## Suggested usage

1. Start each day from `references/day_0X.md`.
2. Use `rag_corpus/` as first local retrieval dataset.
3. Use `qa_lane/` checklists to validate AI outputs and regressions.
